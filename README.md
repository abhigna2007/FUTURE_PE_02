# FUTURE_PE_02 – UGC Ad Content Pack

## 📌 Business Chosen
Glamour Studio – Premium salon in Banjara Hills, Hyderabad.

## ❗ Problem
Traditional ads feel fake and don’t convert well. Customers trust real, relatable content more.

## 💡 Solution
Used AI to generate UGC-style ad scripts that feel natural, emotional, and conversion-focused.

## 🧠 Prompt Logic
- Defined audience clearly
- Used UGC tone (real, casual)
- Followed problem → solution → CTA
- Focused on emotional outcomes

## 🛠 Tools Used
- ChatGPT
- (Optional: CapCut / Lovable)

## 📂 Structure
- prompts/
- outputs/
- assets/

## 🚀 Outcome
Generated ready-to-use ad scripts for Instagram that can be used by brands or agencies.
